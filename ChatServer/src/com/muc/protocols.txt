User and Server


1. User ---> Server

    *login/logoff
    *Status

2. Server ---> User

    *Online/offline

3. User ---> User

    *Direct Messages
    *Broadcast messages / group messaging


Commands:

    *Login <user> <password>
    *Logoff


Direct Messages Format:

    msg <user> <message>


group Messages Format:

    join <topic>
    msg <topic> <msg>
    send: msg <topic
